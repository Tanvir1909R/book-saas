### AI Book Summary Platform (Next.js + MySQL + Prisma + OpenAI)



This project is a modern web application that allows users to **read and listen to AI-generated book summaries**. The platform is built with a full-stack architecture using **Next.js**, **MySQL**, and **Prisma**, with secure authentication and AI integration.



Users can explore book summaries generated by **OpenAI (ChatGPT API)**, making it easy to quickly understand key ideas from books in both **text and audio format**.



#### Key Features:



* AI-powered book summary generation using OpenAI API

* Text-based summaries with clean and responsive UI

* Text-to-audio listening experience for accessibility

* Secure user authentication using JWT

* User management and protected routes

* MySQL database with Prisma ORM for efficient data handling

* Fast and SEO-friendly frontend built with Next.js



#### Tech Stack:



* **Frontend:** Next.js, React

* **Backend:** Next.js API Routes

* **Database:** MySQL

* **ORM:** Prisma

* **Authentication:** JWT

* **AI Integration:** OpenAI API
